Formula Cheat Sheet

Addition/Subtraction (Max)
$\Delta Z = \Delta A + \Delta B$
Addition/Subtraction (Stat)
$\Delta Z = \sqrt{(\Delta A)^2 + (\Delta B)^2}$
Multiplication/Division
$\frac{\Delta Z}{Z} = \frac{\Delta A}{A} + \frac{\Delta B}{B}$
Power Rule
If $Z = A^n$
$\frac{\Delta Z}{Z} = |n|\left(\frac{\Delta A}{A}\right)$
Weighted Mean
$w_i = \frac{1}{(\Delta x_i)^2}$
$x_{best} = \frac{\sum w_i x_i}{\sum w_i}$
Chi-Square
$\chi^2 = \sum \frac{(y_i-y_{fit})^2}{\sigma_i^2}$
